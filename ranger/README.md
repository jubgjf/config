# `ranger` - ranger 配置

## 配置

```shell
ln -s ~/code/config/ranger ~/.config/ranger
```

## 依赖

- ueberzug
- ffmpeg
- ffmpegthumbnailer
- ranger_devicons
  ```shell
  git clone https://github.com/cdump/ranger-devicons2 ~/.config/ranger/plugins/devicons2
  ```
- ranger-zoxide
  ```shell
  git clone https://github.com/jchook/ranger-zoxide.git ~/.config/ranger/plugins/zoxide
  ```
