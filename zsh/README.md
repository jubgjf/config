# `zsh` - zsh 配置

## 依赖

- zsh 样式
  - zsh-autosuggestions
  - zsh-syntax-highlighting
  - starship
- oh-my-zsh 框架
  - oh-my-zsh-git
- 字体
  - nerd-fonts-jetbrains-mono
  - nerd-fonts-cascadia-code
- alias 原应用程序
  - bat
  - htop
  - fd
  - fzf
  - ranger
  - unzip
  - rsync
  - ripgrep-all
  - zoxide
  - exa
  - neovim
- 代码环境
  - anaconda

## 配置

```shell
ln -s ~/code/config/zsh/_zshrc ~/.zshrc
ln -s ~/code/config/zsh/starship.toml ~/.config/starship.toml
```
